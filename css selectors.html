CSS Selectors
CSS Selectors are selects the HTML elements you want to style

CSS Selectors are used to find or select the HTML elements you want to style

We can divided CSS Selectors into five categories

  • Simple Selectors(select elements base on name, id, class)
  • Combination Selectors(select elements base on a specific relationship between them)
  • pseudo-class selectors(select elements base on a certain state)
  • pseudo-elements selectors(select and style a part of an attribute)
  • Attribute selectors(select elemnts based on an attribute or attribute value)
  • Universal selector
  • Elements or Tag selector
  • Class selector
  • id selector
  • Grouping selector
  • Attribute selector
Universal selector
Targets all elements in the Document

Example

*{ margin: 0; padding: 0; }
Class selector
Targets elements with a specific class.

Example

.my class{ color: blue; }
ID selector
targets an element with a specific ID must be unique
Example
# my id{ background color:blue; }
