CSS types
CSS Divided into 3 types they are given below

Inline CSS

Internal CSS

External CSS

Inline CSS
inline CSS invovlves applying styles directly to an HTML element using the styleattribute
Features of Inline CSS
  • Applied directly to individual elements
  • Overrides internal and External CSS if there a conflict
  • Useful for quick changes or testing
Internal CSS
Internal CSS is written within a style block in the section of an HTML element
Features of Internal CSS
  • Applies styles to the entire document.
  • Useful for styling a single web page.
  • Easier to maintain than inline CSS.
Disadvantages
  • Cannot be reused across multiple pages.
  • Increases the size of the HTML file.
External CSS
External CSS is written in a separate file with a .css extension and linked to the HTML document using a tag.
Features of External CSS
  • Styles can be reused across multiple web pages.
  • Promotes separation of content and design.
  • Simplifies maintenance and consistency for large websites.
Disadvantages
  • Requires an extra HTTP request to load the CSS file
